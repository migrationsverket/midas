Release 15

15 november 2024 · 2 min att läsa

Nedan är en sammanfattning av nyheterna i release 15.

Ny komponent: Multiselect​

  • npm
  • Yarn
  • pnpm
npm install @midas-ds/multi-select
yarn add @midas-ds/multi-select
pnpm add @midas-ds/multi-select

<MultiSelect
    label="Favoritfrukter"
    placeholder="Välj dina favoritfrukter"
    items={fruits.map(fruit => {return {id: fruit.value, name: fruit.name}})}
/>

Uppdatering: LinkButton​

LinkButton har en ny, förtydligande design. LinkButton var tydligare grafiskt exakt lik en Button fast som ett <a>-element. Det nya utseendet förtydligar att detta är just en länk. Däremot har den fortfarande varianter och storlekar som en Button.

<LinkButton href="#">Starta tjänst</LinkButton>

Uppdatering: Link​

Komponenten Link är nu uppdaterad med varianterna Fristående och Utsträckt.

Fristående länk​

Fristående länk används när länken ligger utanför ett textstycke.

<>
  <p>Designsystemet utvecklas ständigt. Varannan vecka kommer det ett nytt släpp med buggfixar, nya komponenter eller nya funktioner på befintliga komponenter.</p>
  <Link
      standalone
      href="/blog"
  >
      Läs mer om designsystemets nyheter
  </Link>
</>

Utsträckt länk​

Utsträckt länk används för att göra hela föräldraelementet klickbart.

<div
    style={{
      position: 'relative',
      background: baseColors.gray10,
      padding: '1rem'
    }}
  >
  <p>Designsystemet utvecklas ständigt. Varannan vecka kommer det ett nytt släpp med buggfixar, nya komponenter eller nya funktioner på befintliga komponenter. Om du håller muspekaren här så kan du klicka på länken över hela den gråa ytan.</p>
  <Link
      standalone
      stretched
      href="/blog"
  >
      Läs mer om designsystemets nyheter
  </Link>
</div>
