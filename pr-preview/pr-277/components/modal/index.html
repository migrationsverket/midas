Modal

Modal, Dialog, Dialogruta
Storybook
React Aria

Modaler lägger sig över allt annat innehåll för att fokusera användarens fokus på en sak.

Installation och användning​

  • npm
  • Yarn
  • pnpm
npm install @midas-ds/components
yarn add @midas-ds/components
pnpm add @midas-ds/components
import { Modal } from '@midas-ds/components'

const [isOpen, setIsOpen] = React.useState<boolean>(false)

<>
            <Button onPress={setIsOpen}>Radera frukter</Button>
            <Modal id="modal" title="Vill du radera alla frukter?" isOpen={isOpen} onOpenChange={setIsOpen}>
                <p>Är du säker? De går inte att återskapa.</p>
                <Flex fluid={true}>
                    <FlexItem col='auto'>
                        <Button onPress={() => setIsOpen(false)} variant="danger">Radera frukter</Button>
                    </FlexItem>
                    <FlexItem>
                        <Button
                            onPress={() => setIsOpen(false)}
                            variant='secondary'
                        >
                            Avbryt
                        </Button>
                    </FlexItem>
                </Flex>
            </Modal>
        </>

Avancerat innehåll​

Modaler kan även innehålla mer avancerat innehåll som ett formulär. Använd då autoFocus på första element för att användarens fokus ska flyttas dit när modalen öppnas.

<>
  <Button onPress={setIsOpen}>Öppna modal</Button>
  <Modal id="avsluta" title="Vill du avsluta uppdraget?" isOpen={isOpen} onOpenChange={setIsOpen}>
      <Flex fluid={true} style={{marginBottom: '1rem'}}>
          <FlexItem col>
              <TextArea
                  autoFocus
                  label='Ange anledning'
                  description='Skriv anledning'
                  maxLength={100}
              />
          </FlexItem>
      </Flex>
      <Flex fluid={true}>
          <FlexItem col='auto'>
              <Button onPress={() => setIsOpen(false)}>Spara korrespondens</Button>
          </FlexItem>
          <FlexItem>
              <Button
                  onPress={() => setIsOpen(false)}
                  variant='secondary'
              >
                  Avbryt
              </Button>
          </FlexItem>
      </Flex>
  </Modal>
</>

API​

NamnTypStandardBeskrivning
id *string-Id for accessibility
title *string-Heading for the modal window
isOpen *boolean-Whether the modal is open or not

NamnTypStandardBeskrivning
onOpenChange *(isOpen: boolean) => void-State handler to toggle isOpen
