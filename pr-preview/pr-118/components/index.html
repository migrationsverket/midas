Komponenter

📄️ Accordion

Dragspelskomponenten låter användare visa och dölja delar av relaterat innehåll på en sida.

📄️ Breadcrumbs

Brödsmulor används för att visa användaren vart hen är i ett djupt navigationsträd.

📄️ Button

Knappar används när användaren vill utföra en handling t.ex. spara ifylld information eller öppna ett formulär.

📄️ Card

En card-komponent är en modulär enhet som används för att presentera innehåll på ett strukturerat och visuellt tilltalande sätt.

📄️ Checkbox

Används för att låta användaren välja inget, ett eller flera alternativ.

📄️ ComboBox

Sökbar version av Select

📄️ DatePicker

Fält för att välja ett datum eller ett spann av datum med kalender.

📄️ FileUpload

Filuppladdning används när användaren behöver ladda upp en eller flera filer från det egen filsystemet.

📄️ Flex

Grid baserat på tekniken display flex i css

📄️ Grid

Grid baserat på tekniken display grid i css

📄️ InfoBanner

Information banner visar viktiga och aktuella meddelanden, som till exempel systemnotiser.

📄️ LinkButton

Komponent med samma utseende och beteende som knapp men avsedd att använda som länk internt eller externt i en applikation.

📄️ Link

Komponent som skapar en länk

📄️ Logo

Migrationsverkets officiella logotyp, den röda bron tillsammans med namnet

📄️ Modal

Modal som lägger sig över allt annat innehåll.

📄️ Radio

Radio Button är en typ av inmatningsfält som används för att välja ett alternativ från en uppsättning alternativ

📄️ SearchField

Sökfält är en kombinationskomponent av TextField och Button

📄️ Select

Introduktion

📄️ Spinner

Upplys användaren om att systemet/tjänsten laddar

📄️ Table

En enkel tabell för att visualisera data.

📄️ Tabs

Fördela information över flertalet tabbar.

📄️ TextArea

När användaren ska fylla i längre information

📄️ TextField

Använd ett textfält när användaren ska fylla i en rad med text.

📄️ Toast

Toasts visar korta, tillfälliga meddelanden om åtgärder, fel eller andra händelser i en applikation.
