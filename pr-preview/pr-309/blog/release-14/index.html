Release 14

31 oktober 2024 · 2 min att läsa

Nedan är en sammanfattning av nyheterna i release 14.

Ny komponent: Modal​

Modalen öppnar en ruta ovanför gränssnittet som användaren måste interagera med innan den kan gå vidare. Modalen har flera användningsområden, t.ex. för att förtydliga konsekvenserna av ett val i gränssnittet eller presentera mer information. Läs mer på komponentsidan för Modal.

  • npm
  • Yarn
  • pnpm
npm install @midas-ds/modal
yarn add @midas-ds/modal
pnpm add @midas-ds/modal

<>
  <Button onPress={setIsOpen}>Radera frukter</Button>
  <Modal id="modal" title="Vill du radera alla frukter?" isOpen={isOpen} onOpenChange={setIsOpen}>
      <p>Är du säker? De går inte att återskapa.</p>
      <Flex fluid={true}>
          <FlexItem col='auto'>
              <Button onPress={() => setIsOpen(false)} variant="danger">Radera frukter</Button>
          </FlexItem>
          <FlexItem>
              <Button
                  onPress={() => setIsOpen(false)}
                  variant='secondary'
              >
                  Avbryt
              </Button>
          </FlexItem>
      </Flex>
  </Modal>

</>

Ikoner på Button​

Button kan nu acceptera ikoner som prop. Ikonen läggs till via icon. Du kan välja vilken sida av texten ikonen ska vara på med iconPlacement.

<Button
  icon={Plus}
  variant='tertiary'
  iconPlacement="right"
>
  {"Lägg till rad"}
</Button>

Mindre justeringar​

  • Varningsmeddelanden på samtliga inputfält är ny röda för tydlighetens skull.
  • InfoBanner beter sig nu som en vanlig <div>
  • LinkButton har nu samma egenskaper som Button
