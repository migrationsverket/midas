Introduktion​

En MultiSelect visar en hopfällbar lista med alternativ och låter en användare välja ett eller flera av dem.

MultiSelect är en typ av inmatningsfält som används för att välja ett eller flera alternativ från en uppsättning av flera alternativ. Om det ska gå att välja flera eller inget alternativ men är det Checkbox som ska användas Om det är färre alternativ än fem används med fördel Radio istället.

Installation och användning​

  • npm
  • Yarn
  • pnpm
npm install @midas-ds/select
yarn add @midas-ds/select
pnpm add @midas-ds/select
import { MultiSelect } from '@midas-ds/multi-select'
