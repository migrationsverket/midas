Release 18

16 januari 2025 · En minut att läsa

Nedan är en sammanfattning av nyheterna i release 18.

API på komponenter är tillbaka​

Nu finns efter en tids uppehåll API och props för vardera komponent beskrivna på komponentsidorna. Dessa kommer fortsätta förbättras i nästa release.

Instruktioner för klientnavigering (Client Side Routing)​

Tydligare instruktioner och fler möjligheter för att integrera komponenterna med din router finns nu på denna sida

Nya varianter på Logo​

Komponenten Logo har fått nya storlekar och varianter.

<div style={{display: 'flex', flexWrap: 'wrap', gap: '1rem'}}>
  <Logo size="x-small" />
  <Logo size="small" />
  <Logo size="medium" />
  <Logo size="large" />
</div>

Sök​

Du kan nu söka på denna webplats. Du finner söken högst upp i högra hörnet eller med kortkommandot ctrl + K alternativt på Mac ⌘ + K.

Mindre justeringar​

  • Justerat och förbättrat hur Modal växer beroende på innehållet.
