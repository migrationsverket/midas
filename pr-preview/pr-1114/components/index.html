Komponenter

Alla komponenter i Midas designsystem.

AccordionDragspelskomponenten låter användare visa och dölja delar av relaterat innehåll på en sida.
BadgeBadge används tillsammans med en ikon eller knapp för att påvisa att användaren har något oläst att ta del av.
BreadcrumbsBrödsmulor används för att visa användaren vart hen är i ett navigationsträd.
ButtonKnappar används när användaren vill utföra en handling t.ex. spara ifylld information eller öppna ett formulär.
CalendarAnvänd en kalender när användaren ska välja ett datum eller visualisera ett datum valt på annat sätt.
CardEtt card är en modulär komponent som används för att presentera innehåll på ett strukturerat och visuellt tilltalande sätt.
CheckboxAnvänds för att låta användaren välja inget, ett eller flera alternativ.
ColorSchemeSwitchVäxlare för färgschema (ljust, mörkt, systemets inställning)
ComboBoxSökbar version av Select
DatePickerFält för att välja ett datum eller ett spann av datum med kalender.
DateFieldAnvänd ett datumfält när användaren ska fylla i ett datum.
FileListLista uppladdade filer
FileTriggerÅtkomst till filsystemet
GridGrid baserat på tekniken display flex i css
HeadingRubrikkomponent med semantiska nivåer
InfoBannerInformation banner visar viktiga och aktuella meddelanden, som till exempel systemnotiser.
LayoutEn skalkomponent för att få en simpel sidomeny och sidhuvud.
LinkButtonKomponent med samma utseende och beteende som knapp men avsedd att använda som länk internt eller externt i en applikation.
LinkKomponent som skapar en länk
LogoMigrationsverkets officiella logotyp, den röda bron tillsammans med namnet
MenuEn meny kan användas för att samla ihop funktioner som sällan används eller som kompletterar nuvarande funktion.
ModalModal som lägger sig över allt annat innehåll.
PaginationPaginering för tabeller
PopoverPopover kan användas för att förklara en knapp eller annan interaktion när det inte går att få plats med informationen i knappen.
ProgressBarVisar framsteg för en pågående process
RadioRadio är en typ av inmatningsfält som används för att välja ett alternativ från en uppsättning alternativ
SearchFieldSökfält är ett inmatningsfält anpassat för sökning
SelectVälj ett alternativ från en lista
SkeletonLaddningsindikator för komponenter
SpinnerUpplys användaren om att systemet/tjänsten laddar
TableEn enkel tabell för att visualisera data.
TabsOrganisera information med hjälp av flikar.
TagStatusindikator
TextBrödtext och löpande textinnehåll
TextAreaNär användaren ska fylla i längre information
TextFieldAnvänd ett textfält när användaren ska fylla i en rad med text.
ToastToasts visar korta, tillfälliga meddelanden om åtgärder, fel eller andra händelser i en applikation.
TooltipTooltip kan användas för att förklara en knapp eller annan interaktion när det inte går att få plats med informationen i knappen.
