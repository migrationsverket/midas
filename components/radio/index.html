Radio

Radioknappar
Storybook
React Aria

Inmatningsfält som används för att välja exakt ett alternativ.

import { Radio, RadioGroup } from '@midas-ds/components'
<RadioGroup
  label='Välj din favoritfrukt'
  description='Du kan bara välja en'
>
  <Radio value='apple'>Äpple</Radio>
  <Radio value='banan'>Banan</Radio>
  <Radio value='kiwi'>Kiwi</Radio>
  <Radio value='apelsin'>Apelsin</Radio>
</RadioGroup>
Du kan bara välja en

Kontrollerat val​

Använd attributet value och eventet onChange för att använda controlled value på RadioGroup. Den valda radioknappens värde skickas till callbacken, så kan du använda det för att uppdatera ditt state.

Notera att ett tomt val representeras av null eller undefined och inte en tom sträng.

import React from 'react'
const [selectedFruit, setSelectedFruit] = useState<string | null>(null)

return (
  <RadioGroup
    label='Välj din favoritfrukt'
    description='Du kan bara välja en'
    value={selectedFruit}
    onChange={setSelectedFruit}
  >
    <Radio value='apple'>Äpple</Radio>
    <Radio value='banan'>Banan</Radio>
    <Radio value='kiwi'>Kiwi</Radio>
    <Radio value='apelsin'>Apelsin</Radio>
  </RadioGroup>
)
Du kan bara välja en

Riktlinjer​

  • Fältetiketten ska inledas med stor bokstav och inte följas av punkt.
  • Lägg valen i bokstavsordning om ingen annan given ordning är mer lämplig, t.ex. det mest populära valet först.
  • Radioknapparna placeras som regel vertikalt för att underlätta avläsning. Om antalet alternativ för en given fråga är begränsat till två och det är ont om vertikalt utrymme så kan en horisontell orientering användas.
Max 2 val vid horisontellt läge

Mönster​

Det finns två mönster för Radioknappar:

  1. Inget alternativ är valt från början. Detta alternativ kräver att användaren tar ställning för att kunna gå vidare, vilket i de flesta fall är lämpligt. Välj detta mönster om det är svårt att förutse eller olämpligt att ge ett förval.
Du kan bara välja en

Om användaren behöver ha möjlighet att avstå från att välja ett de givna alternativen är rekommendationen att skapa ett sista alternativ som du kallar "Inget av givna alternativ" eller liknande.

Du kan bara välja en
  1. Ett av alternativen är valt från början. Välj detta mönster om det finns ett val som är förväntat eller önskvärt och det inte ger stora konsekvenser för användaren att inte ta ställning.
Det blir gott!

Val av komponent​

Använd Radioknappar när användaren bara ska göra ett av flera val i en lista. Använd Checkbox om flera val ska kunna göras. Om det är fler alternativ än fem bör Select användas istället.
