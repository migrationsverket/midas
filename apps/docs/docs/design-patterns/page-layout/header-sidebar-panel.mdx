# Header, Sidebar och Panel

Kombinationen [Header](/dev/layout/components/header), [Sidebar](/dev/layout/components/sidebar) och [Panel](/dev/layout/components/panel) innehåller alla komponenter för desktopapplikationer. Det är det vanligaste för våra mer avancerade applikationer, där Panel används för att visa kontextuell information t.ex. notifieringar eller mer information om element i en lista.

## Navigation

När fönstret eller skärmen är under 640px i bredd göms Sidebar automatiskt. För att stödja navigation för alla fönster- och skärmstorlekar måste navigation via [MobileMenu](/dev/layout/components/header/#mobilemenu) alltid inkluderas i trädet. `MobileMenu` renderar ingenting på desktop eftersom den är villkorsstyrd via `useIsMobileDevice()`. För att undvika kodduplicering rekommenderas att lägga navigationsinnehållet i en delad komponent och använda den till MobileMenu och Sidebar.

```tsx
import {
  Layout,
  LayoutContent,
  Main,
  Header,
  HeaderActions,
  HeaderAction,
  Sidebar,
  Navigation,
  NavigationItem,
  NavigationLink,
  PanelProvider,
  PanelRegion,
} from '@midas-ds/layout'
import { House, Settings, Bell, User } from 'lucide-react'

export default function App() {
  return (
    <PanelProvider panelBehavior='replace'>
      <Layout>
        <Header>
          <MobileMenu title='Meny'>
            <Navigation>
              <NavigationItem>
                <NavigationLink
                  href='/'
                  icon={<House />}
                  isActive
                >
                  Hem
                </NavigationLink>
              </NavigationItem>
              <NavigationItem>
                <NavigationLink
                  href='/installningar'
                  icon={<Settings />}
                >
                  Inställningar
                </NavigationLink>
              </NavigationItem>
            </Navigation>
          </MobileMenu>
          <HeaderActions>
            <HeaderAction icon={<Bell />}>Notifieringar</HeaderAction>
            <HeaderAction icon={<User />}>Min profil</HeaderAction>
          </HeaderActions>
        </Header>

        <LayoutContent>
          <Sidebar title='Navigation'>
            <Navigation>
              <NavigationItem>
                <NavigationLink
                  href='/'
                  icon={<House />}
                  isActive
                >
                  Hem
                </NavigationLink>
              </NavigationItem>
              <NavigationItem>
                <NavigationLink
                  href='/installningar'
                  icon={<Settings />}
                >
                  Inställningar
                </NavigationLink>
              </NavigationItem>
            </Navigation>
          </Sidebar>

          <Main>{/* Ditt innehåll */}</Main>
          <PanelRegion />
        </LayoutContent>
      </Layout>
    </PanelProvider>
  )
}
```

</div>
