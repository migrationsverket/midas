# Laddningsindikatorer

import { Skeleton, Spinner } from '@midas-ds/components'
import { ButtonLoading } from '@site/src/components/examples/loading/LoadingExamples'

Laddningsindikatorer är visuella platshållare för komponenter eller information som visas medan de laddas.
Vi använder oss av två olika laddningsindikator: [Skeleton](/components/skeleton) och [Spinner](/components/spinner).

Vi använder i regel Skeleton som laddningsindikator. Undantagen är 1) när sidan laddar utan att någon ändras eller 2) när du inte vet hur gränssnittet kommer att se ut efter att det har laddat klart

## Skeleton

Skeleton är vår primära laddningsindikator. Den är en minimal representation av de komponenter som kommer finnas i gränssnittet, vilket ger användaren en bra förståelse för hur det färdiga gränssnittet kommer vara strukturerat.

```tsx
<Skeleton height='48px' />
```

<div className='card'>
  <Skeleton height='48px' />
</div>

## Spinner i Button

Används när användaren via knappen triggar en process som kräver att systemet laddaar en kortare tid, tex när ett formulär skickas in.

<ButtonLoading />

### Riktlinjer

- Visa spinner i knappen, till vänster om etiketten.
- Låt knappen vara enabled men blockera interaktion
- Notifiera användaren när systemet laddat klart, tex med en toast

## Fullskärmsspinner

Används primärt när gränssnittet inte ändras, till exempel när en fil laddas upp i bakgrunden. Om det väntas ta mer än 10 sekunder att ladda bör det finnas ett meddelande till användaren om det.

```tsx
<Spinner />
```

<div className='card'>
  <Spinner />
</div>
