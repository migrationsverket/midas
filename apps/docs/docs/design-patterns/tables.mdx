# Tabeller

## Filtrering

### Inline-filter

Inline-filter innebär att filtren placeras direkt ovanför tabellen och är alltid synliga. Detta gör det enkelt att upptäcka vilka filter som finns tillgängliga, se vilka filter som redan är aktiva samt justera flera filter i följd utan extra interaktioner. Filtren kan vid behov läggas i en [Accordion](/components/accordion) för att kunna fällas ihop och ta mindre plats.

Använd detta mönster när

- du har ett fåtal filter
- användaren snabbt behöver kunna justera flera filter på en gång
- det är viktigt att direkt kunna se vilka filter som är aktiva

### Filter i popover

I detta mönster öppnas filtrvalen i en popover via en knapp eller en ikon. Det ger en fokuserad tabellvy när filtrering inte är användarens primära handling. Inne i popovern läggs filtren i [Accordion](/components/accordion) som kan expanderas och fällas ihop, vilket möjliggör en tydlig och skalbar struktur även när antalet filter är stort.

- datatunga applikationer där användaren ibland behöver göra mer detaljerade avgränsningar
- filtreringen inte är en primär uppgift för de flesta användare
- det finns många filter
- du vill kunna gruppera filter
