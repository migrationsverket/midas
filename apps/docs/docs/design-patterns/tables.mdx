# Tabeller

import useBaseUrl from '@docusaurus/useBaseUrl'
import ThemedImage from '@theme/ThemedImage'

Tabeller används för att presentera strukturerad data på ett tydligt och jämförbart sätt. För att stödja olika användarbehov och datamängder innehåller designsystemet ett antal etablerade mönster för tabeller.

## Val av kolumner

För tabeller med många kolumner kan det vara lämpligt att låta användaren styra vilka kolumner som ska visas. Detta gör vi genom att ha en knapp placerad i anslutning till tabellen som öppnar en popover. Popovern innehåller en lista med kryssrutor där varje kryssruta representerar en kolumn i tabellen. Genom att markera eller avmarkera kryssrutorna kan användaren välja vilka kolumner som ska vara synliga.
