## Sammanfattning

Vid valideringsfel visas en sammanfattning av de fel som förekommer högst upp på sidan. Sammanfattningen ska bestå av förklaring
på vad som är fel och konkreta råd på hur det/de går att åtgärda för att undvika nya fel. Sammanfattningen går endast att ta bort genom att åtgärda de fel som anges. 


## Felmeddelande

När det sker ett fel eller om det saknas obligatorisk information visas ett felmeddelande under komponenten. Meddelandet ska förklara
vad som är fel och hur det går att åtgärda. Felmeddelandet skrivs med text i rött och inleds med en varningstriangel.
