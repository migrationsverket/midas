Midas kan installeras och användas i ett React-projekt med en pakethanterare som till exempel npm eller yarn. Alla komponenter är inkluderade i ett paket för att underlätta installation och uppdatering av komponentbiblioteket.

```bash npm2yarn
npm install @midas-ds/components
```

## Importera och använd komponent

Generellt, importera och använd direkt:

```tsx {1,6}
import { Button } from '@midas-ds/components'

export default function App({ children }) {
  return (
    <main>
      <Button variant='primary'>Klicka här</Button>
      {children}
    </main>
  )
}

export default App
```

För specifika instruktioner, se respektive komponent på våra [komponentsidor](/components/accordion) eller utforska komponenterna mer i vår Storybook.

## Använda tokens/theme

Importera `theme.css` i roten av din applikation:

```tsx title="main.tsx"
import '@midas-ds/components/theme.css'
```

Använd Midas tokens (läs mer om våra [tokens](/basics/tokens/)) som variabler i css:

```css title="App.css"
.kaffekopp {
  border: 1px solid var(--midas-border-color-primary);
  background-color: var(--midas-button-background-primary);
  color: var(--midas-text-on-color);
  padding: var(--midas-spacing-30);
  width: fit-content;

  &:hover {
    background-color: var(--midas-button-background-primary-hover);
  }
}
```

```tsx title="App.tsx"
import 'App.css'

export default function App() {
  return <div className={'kaffekopp'}>Colombiansk bönjuice</div>
}

export default App
```

## Installera Inter som typsnitt

Detta är ett rekommenderat sätt att installera vårt typsnitt Inter.

```tsx {1} title="App.tsx (rootfilen i din app)"
import '@midas-ds/components/global.css'
import { Button } from '@midas-ds/components'

export default function App({ children }) {
  return (
    <main>
      <Button variant='primary'>Klicka här</Button>
      {children}
    </main>
  )
}

export default App
```

Det går självklart att använda andra metoder så länge vikterna 400, 500 och 600 inkluderas samt att det importerade typsnittet heter just "Inter" för att komponenterna ska ta användning av det.

Mer info finns på [GitHub](https://github.com/migrationsverket/midas) samt under [guider](/dev/common-issues)

## Dependencies

Midas bygger på flertalet beroenden som kan vara bra att känna till. Förutom att vara ett komponentbibliotek baserat på React, så innehåller Midas även verktyg och bibliotek som underlättar utveckling och säkerställer tillgänglighet.

- [React Aria](https://react-spectrum.adobe.com/react-aria/) - Alla komponenter med några få undantag baseras på React Arias bibliotek för att säkerställa tillgänglighet och användarvänlighet. Använder du en komponent från Midas kan det även vara värt att läsa React Arias motsvarande dokumentation. Dessa finns länkade på komponentsidan.
- [Lucide Icons](https://lucide.dev/) - Midas använder Lucide Icons för att tillhandahålla ikoner i komponenter.
