This page lists all available design tokens in the Midas Design System. These tokens can be imported and used in your applications via CSS variables or JavaScript.

## Typography

Typography tokens define text styles including font families, sizes, weights, line heights, and composite typography styles.

### Composite Typography

Composite typography tokens combine multiple properties into a single token following the W3C Design Tokens format.

<TokenList tokens={{
  body: tokenDictionary.typography.body,
  'body-small': tokenDictionary.typography['body-small'],
  description: tokenDictionary.typography.description,
  'description-small': tokenDictionary.typography['description-small']
}} />
