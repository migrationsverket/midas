Informationsmeddelande visar viktiga och aktuella meddelanden om händelser eller förändringar, som till exempel systemnotiser.
Den används för att ge användaren feedback, exempelvis vid fel, varningar eller bekräftelser och är
utformad för att vara tydlig och lätt att förstå utan att störa användarens flöde.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/info-banner
```

```tsx
import { InfoBanner } from '@midas-ds/info-banner'
```

## Exempel

<MidasTabs
  label={'Välj typ'}
  tabs={['Info', 'Important', 'Warning', 'Success']}
>
  <InfoBanner
    type={'info'}
    title={'Information'}
    message={
      'Detta är ett informationsmeddelande om att informationsmeddelande visar statisk information, till exempel systemnotiser.'
    }
  />
  <InfoBanner
    type={'important'}
    title={'Du behöver komplettera'}
    message={'Visas för användaren för att uppmana till att något måste göras.'}
  />
  <InfoBanner
    type={'warning'}
    title={'Formuläret har inte skickats'}
    message={'Visas för användaren för att bekräfta att en action gått snett.'}
  />
  <InfoBanner
    type={'success'}
    dismissable={true}
    title={'Formuläret har skickats'}
    message={'Visas för användaren för att bekräfta att en action gått rätt till.'}
  />
</MidasTabs>
