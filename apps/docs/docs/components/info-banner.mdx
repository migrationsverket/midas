## Dismissable

Det går att göra det möjligt för användaren att stänga informationsmeddelandet via `dismissable`. Det bör inte användas för varningar.

```tsx
<InfoBanner
  dismissable
  type={'success'}
  title={'Det gick bra!'}
  message={'Detta är ett meddelande om att allt gick bra.'}
/>
```
