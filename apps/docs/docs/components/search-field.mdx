Inmatningsfält anpassat för sökning.

```tsx
import { SearchField } from '@midas-ds/components'
```

:::warning Deprecated: inbyggd sökknapp

`buttonText` och den inbyggda sökknappen är deprecated sedan v17.9.0 och tas bort i nästa major. Komponera en egen `Button` utanför fältet istället — se exemplen nedan.

:::

## Grundläggande användning

Utan en explicit knapp kan fältet användas för realtidsfiltrering via `onChange`, eller för sök-on-submit via `onSubmit` — Enter-tangenten fungerar alltid. Lägg till en knapp (se nedan) enbart när ett explicit submit-steg behövs.

```tsx
{/* Real-time filter */}
<SearchField placeholder='Filtrera efter namn' onChange={filterResults} />

{/* Keyboard submit (Enter) — no button needed */}
<SearchField placeholder='Sök efter frukter' onSubmit={handleSubmit} />
```

## Sök med knapp

Använd ett externt `Button`-element när sökningen ska triggas explicit — t.ex. vid en söksida med serveranrop. Placera knappen bredvid fältet.

```tsx
const [query, setQuery] = React.useState('')

<SearchField
  placeholder='Sök efter frukter'
  onChange={setQuery}
  onSubmit={handleSubmit}
/>
<Button onPress={() => handleSubmit(query)}>Sök</Button>
```

export const SearchWithButton = () => {
  const [query, setQuery] = React.useState('')
  return (
    <div style={{ display: 'flex', gap: '8px', alignItems: 'flex-start' }}>
      <SearchField
        placeholder='Sök efter frukter'
        onChange={setQuery}
        onSubmit={(v) => console.log('submit:', v)}
      />
      <Button onPress={() => console.log('submit:', query)}>Sök</Button>
    </div>
  )
}

<div className='card'>
  <SearchWithButton />
</div>
