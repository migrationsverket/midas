Sökfält är ett inmatningsfält anpassat för sökning. Sökfältet fungerar med tangentbord där `Enter` fungerar
som submit och `ESC` rensar sökfältet.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/search-field
```

```tsx
import { SearchField } from '@midas-ds/search-field'
```

<LiveCodeBlock scope={{ SearchField }}>{`
<SearchField placeholder="Sök efter frukter"/>`}
</LiveCodeBlock>
