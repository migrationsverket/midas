Komponent för att visa beskrivande etiketter som kan användas för att kategorisera information.

```tsx
import { Tag, TagGroup } from '@midas-ds/components'
```

```tsx
<TagGroup>
  <Tag>Tag med information</Tag>
</TagGroup>
```

<div className='card'>
  <TagGroup aria-label='Taggar'>
    <Tag>Tag med information</Tag>
  </TagGroup>
</div>

## Användning

Tag används som en del av [Select](../select#flerval) om `showTags` är aktiverat men kan också användas fristående eller inuti andra komponenter.

```tsx
<Card horizontal>
  <CardContent>
    <CardTitle>Köp frukt</CardTitle>
    <Text>Vi behöver köpa frukt till fika</Text>
    <TagGroup>
      <Tag type='info'>Inte påbörjat</Tag>
    </TagGroup>
  </CardContent>
  <CardActions>
    <Button variant='tertiary'>Utför uppgift</Button>
  </CardActions>
</Card>
```

<div className='card'>
  <Card horizontal>
    <CardContent>
      <CardTitle>Köp frukt</CardTitle>

  </Card>
</div>

## Varianter

Tag finns i fem varianter.

```tsx
<TagGroup>
  <Tag>Default</Tag>
  <Tag type='success'>Success</Tag>
  <Tag type='info'>Info</Tag>
  <Tag type='important'>Important</Tag>
  <Tag type='warning'>Warning</Tag>
</TagGroup>
```

<div className='card'>
  <TagGroup aria-label='Taggar'>
    <Tag>Default</Tag>
    <Tag type='success'>Success</Tag>
    <Tag type='info'>Info</Tag>
    <Tag type='important'>Important</Tag>
    <Tag type='warning'>Warning</Tag>
  </TagGroup>
</div>

## Dismissable

Det går att göra det möjligt för användaren att stänga tag via `dismissable`.

```tsx
<TagGroup>
  <Tag dismissable>Default</Tag>
</TagGroup>
```

<div className='card'>
  <TagGroup aria-label='Taggar'>
    <Tag dismissable>Default</Tag>
  </TagGroup>
</div>
