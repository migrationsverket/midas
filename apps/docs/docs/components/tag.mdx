Komponent för att visa beskrivande etiketter som kan användas för att kategorisera information.

```tsx
import { Tag, TagGroup, TagList } from '@midas-ds/components'
```

```tsx
<TagGroup aria-label='Taggar'>
  <TagList>
    <Tag>Tag med information</Tag>
  </TagList>
</TagGroup>
```

<div className='card'>
  <TagGroup aria-label='Taggar'>
    <TagList>
      <Tag>Tag med information</Tag>
    </TagList>
  </TagGroup>
</div>

## Användning

Tag används som en del av [Select](/components/select) om `showTags` är aktiverat men kan också användas fristående eller inuti andra komponenter.

```tsx
<Card horizontal>
  <CardContent>
    <CardTitle>Köp frukt</CardTitle>
    <Text>Vi behöver köpa frukt till fika</Text>
    <TagGroup aria-label='Taggar'>
      <TagList>
        <Tag color='blue'>Inte påbörjat</Tag>
      </TagList>
    </TagGroup>
  </CardContent>
  <CardActions>
    <Button variant='tertiary'>Utför uppgift</Button>
  </CardActions>
</Card>
```

<div className='card'>
  <Card horizontal>
    <CardContent>
      <CardTitle>Köp frukt</CardTitle>
      <Text>Vi behöver köpa frukt till fika</Text>
      <TagGroup aria-label='Taggar'>

          <Tag color='blue'>Inte påbörjat</Tag>
        </TagList>
      </TagGroup>
    </CardContent>
    <CardActions>
      <Button variant='tertiary'>Utför uppgift</Button>
    </CardActions>
  </Card>
</div>

## Färger

Tag finns i fem färger.

```tsx
<TagGroup aria-label='Färger'>
  <TagList>
    <Tag>Default</Tag>
    <Tag color='green'>Green</Tag>
    <Tag color='blue'>Blue</Tag>
    <Tag color='yellow'>Yellow</Tag>
    <Tag color='red'>Red</Tag>
  </TagList>
</TagGroup>
```

<div className='card'>
  <TagGroup aria-label='Färger'>
    <TagList>
      <Tag>Default</Tag>
      <Tag color='green'>Green</Tag>
      <Tag color='blue'>Blue</Tag>
      <Tag color='yellow'>Yellow</Tag>
      <Tag color='red'>Red</Tag>
    </TagList>
  </TagGroup>
</div>

## Dismissable

Det går att göra det möjligt för användaren att stänga tag via `isDismissable`.

```tsx
<TagGroup aria-label='Taggar'>
  <TagList>
    <Tag isDismissable>Default</Tag>
  </TagList>
</TagGroup>
```

<div className='card'>
  <TagGroup aria-label='Taggar'>
    <TagList>
      <Tag isDismissable>Default</Tag>
    </TagList>
  </TagGroup>
</div>

## Ikoner

Ikoner kan användas inuti en tag.

```tsx
<TagGroup aria-label='Taggar'>
  <TagList>
    <Tag>
      <ScooterIcon /> Scooters
    </Tag>
  </TagList>
</TagGroup>
```

<div className='card'>
  <TagGroup aria-label='Taggar'>
    <TagList>
      <Tag>
        <ScooterIcon /> {'Scooters'}
      </Tag>
    </TagList>
  </TagGroup>
</div>

## Statiskt och dynamisk innehåll

`TagList` använder React arias API för [Collections](https://react-aria.adobe.com/collections) och kan användas dynamiskt.

```tsx
import { Tag, TagGroup, TagList } from '@midas-ds/components'

const DynamicExample = () => {
  const items = [
    { id: 'green', name: 'Green' },
    { id: 'blue', name: 'Blue' },
    { id: 'yellow', name: 'Yellow' },
    { id: 'red', name: 'Red' },
  ]

  return (
    <TagGroup aria-label='Taggar'>
      <TagList items={items}>
        {item => (
          <Tag
            id={item.id}
            color={item.id}
          >
            {item.name}
          </Tag>
        )}
      </TagList>
    </TagGroup>
  )
}
```
