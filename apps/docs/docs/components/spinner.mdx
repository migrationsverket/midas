## Installation

```bash npm2yarn
npm install @midas-ds/components
```

```tsx
import { Spinner } from '@midas-ds/components'
```

## Varianter

### Fristående spinner

<LiveCodeBlock scope={{ Spinner }}>{`<Spinner />`}</LiveCodeBlock>

### Kombinerad med andra komponenter

Använd varianten `small` för att kombinera med andra komponenter t.ex. Button

<LiveCodeBlock scope={{ Spinner, Button }}>
  {`<Button>
  <Spinner
    dark
    small
  />
  Laddar
</Button>`}
</LiveCodeBlock>

## Riktlinjer

Om det väntas ta mer än 10 sekunder att ladda sidan bör det förutom en spinner finnas ett meddelande till användaren om det.

För val av laddningsindikator se <Link href="/design-patterns/page-loading" >{'Mönster för laddningsindikatorer'}</Link>
