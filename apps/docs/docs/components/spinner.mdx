Komponenten används för att upplysa användaren om att systemet/tjänsten laddar eller jobbar på något i bakgrunden.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/spinner
```

```tsx
import { Spinner } from '@midas-ds/spinner'
```

<LiveCodeBlock scope={{ Spinner }}>{`<Spinner />`}</LiveCodeBlock>

### Kombinerat med andra komponenter

Använd varianten small för att kombinera med andra komponenter t.ex Button

<LiveCodeBlock scope={{ Spinner, Button }}>
  {`<Button>
  <Spinner
    dark
    small
  />
  Laddar
</Button>`}
</LiveCodeBlock>

## Riktlinjer

Om det väntas ta mer än 10 sekunder att ladda sidan bör det förutom en spinner finnas ett meddelande till användaren om det.
