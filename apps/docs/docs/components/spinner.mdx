Komponent som används för att upplysa användaren om att systemet/tjänsten laddar eller jobbar på något i bakgrunden.
Läs mer om hur spinner används i [Mönster för laddningsindikatorer](/design-patterns/page-loading)

```tsx
import { Spinner } from '@midas-ds/components'

<Spinner/>
```

<div className='card'>
  <Spinner/>
</div>

## Varianter

### Storlek

Standardstorleken är helskärmsspinner. Använd varianten `small` när spinner ligger bredvid andra komponenter.

```tsx
<Spinner small/>
```

<div className='card'>
  <Spinner
    small
  />
</div>

### Färg

Spinnern ärver sin färg från textfärgen på det element den placeras i.

```tsx
<Spinner/>
```

<div className='card'>
  <Spinner/>
</div>

### Valfri färg

```tsx
<Spinner color={variables.iconOnColor} small/>
```

<div
  className='card'
  style={{ backgroundColor: variables.colorGray200 }}
>
  <Spinner color={variables.iconOnColor} small/>
</div>

### Knapp med spinner

För att visa en spinner i Button används property `isPending` på [Button](/components/button)

```tsx
<Button isPending>
  <Spinner small/>
  Laddar...
</Button>
```

<Button isPending><Spinner small/>Laddar...</Button>

## Riktlinjer

Om det väntas ta mer än 10 sekunder att ladda sidan bör det förutom en spinner finnas ett meddelande till användaren om det.
