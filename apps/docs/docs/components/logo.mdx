På webbsida och e-tjänster ska Migrationsverkets officiella logotyp, den röda bron tillsammans med namnet, finnas.

```tsx
<Logo />
```

<div className='card'>
  <Logo />
</div>

## Installation

```bash npm2yarn
npm install @midas-ds/components
```

```tsx
import { Logo } from '@midas-ds/components'
```

## Varianter

Logotypen finns i fyra olika storlekar (xsmall, small, medium och large) samt med eller utan bakgrund. X-small skiljer sig från de andra då det endast är ett M. Den används när skärmstorleken är mindre än 380px för spara plats i headern.

### X-small

```tsx
<Logo size="x-small" />
<Logo primary={false} size="x-small" />
```

<div
  className='card'
  style={{ flexWrap: 'wrap', gap: '1rem', flexDirection: 'row' }}
>
  <Logo size='x-small' />
  <Logo
    primary={false}
    size='x-small'
  />
</div>
