## Introduktion

På webbsida och e-tjänster ska Migrationsverkets officiella logotyp, den röda bron tillsammans med namnet, finnas.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/logo
```

```tsx
import { Logo } from '@midas-ds/logo';
```

```tsx live
<>
    <Logo/>
    <Logo background/>
    <Logo dark/>
    <Logo dark background/>
    <Logo size={'small'}/>
</>
```
