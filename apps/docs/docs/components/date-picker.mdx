Inmatningsfält för att välja ett eller flera datum. Består av två komponenter `DatePicker` för att välja ett datum samt `DateRangePicker` för att välja ett spann mellan två datum.

<LiveCodeBlock hideCode scope={{ DatePicker }}>{`<DatePicker label="Välj ett datum" aria-label="Välj datum" />`}</LiveCodeBlock>

## Installation

```bash npm2yarn
npm install @midas-ds/components
```

```tsx
import { DatePicker } from '@midas-ds/components'
```

```tsx
import { DateRangePicker } from '@midas-ds/components'
```
