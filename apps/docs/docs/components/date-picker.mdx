Fält för att välja ett datum eller ett spann av datum med kalender. Består av två komponenter `DatePicker` för att välja ett datum samt `DateRangePicker` för att välja ett spann av (2) datum.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/date-picker
```

### DatePicker

```tsx
import { DatePicker } from '@midas-ds/components'
```

<LiveCodeBlock scope={{ DatePicker }}>{`<DatePicker aria-label="Välj datum" />`}</LiveCodeBlock>

### DateRangePicker

```tsx
import { DateRangePicker } from '@midas-ds/components'
```

<LiveCodeBlock scope={{ DateRangePicker }}>{`<DateRangePicker aria-label="Välj spann av datum" />`}</LiveCodeBlock>
