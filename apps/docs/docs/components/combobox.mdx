## Introduktion

En ComboBox (kombination av Select och TextField) visar en hopfällbar lista med alternativ och låter en användare söka bland dem och välja ett av dem.

ComboBox är en typ av input-fält som används för att välja **ett** alternativ av fler än fyra (4) av flera alternativ. Om det ska gå att välja flera eller inget alternativ är det [Checkbox](./checkbox.mdx) som ska användas
Om det är fler alternativ än färre än fyra (4) används med fördel [Radio](./radio.mdx) istället.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/combobox
```

```tsx
import { ComboBox } from '@midas-ds/combobox'
```
