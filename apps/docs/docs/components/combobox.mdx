En ComboBox är en sökbar version av Select. Combobox används för att välja **ett** alternativ av fler än fyra (4) alternativ. 

<LiveCodeBlock hideCode scope={{ ComboBox, ComboBoxItem }}>
  {`<ComboBox
    label="Vad är din favoritfrukt?"
    placeholder="Välj en frukt"
    items={fruits.map(fruit => {return {id: fruit.value, name: fruit.name}})}
>
    {(item) => <ComboBoxItem>{item.name}</ComboBoxItem>}
</ComboBox>`}
</LiveCodeBlock>

## Installation 

```bash npm2yarn
npm install @midas-ds/components
```

```tsx
import { ComboBox } from '@midas-ds/components'
```



## Riktlinjer

### Val av komponent

- Om det ska gå att välja flera eller inget alternativ är det [Checkbox](./checkbox.mdx) eller [MultiSelect](./multi-select.mdx) som ska användas. 
- Om det är färre alternativ än fyra (4) används med fördel [Radio](./radio.mdx) istället.
