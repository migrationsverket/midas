## Introduktion

Använd ett textfält när användaren ska fylla i längre information tex en beskrivning, kommentar, fritext osv. För kortare, striktare input används [TextField](./textfield.mdx).

## Installation och användnin

```bash npm2yarn
npm install @midas-ds/textarea
```

```tsx
import { TextArea } from '@midas-ds/textarea'
```
