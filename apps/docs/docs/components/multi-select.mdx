## Introduktion

MultiSelect är en typ av inmatningsfält som används för att välja **ett eller flera** alternativ från en längre lista. Om det är endast är ett fåtal alternativ kan [Checkbox](./checkbox.mdx) användas istället.


## Installation och användning

```bash npm2yarn
npm install @midas-ds/select
```

```tsx
import { MultiSelect } from '@midas-ds/multi-select'
```
