## Introduktion

Använd ett inmatningsfält när användaren ska fylla i kortare information, tex namn, personnummer eller epostadress. För längre input används [TextArea](./textarea.mdx).

## Installation och användning

```bash npm2yarn
npm install @midas-ds/textfield
```

```tsx
import { TextField } from '@midas-ds/textfield';
```

<LiveCodeBlock scope={{TextField}}>
{`<TextField 
  label={'Etikett'}
  description={'Beskrivning'}
/>`}
</LiveCodeBlock>

### Validering

TextField validerar automatiskt på `type` och `isRequired`

<LiveCodeBlock scope={{TextField}}>
{`<TextField
  label={'Skriv in din e-post'}
  description={'Obligatorisk!'}
  isRequired
  type={'email'}
/>`}
</LiveCodeBlock>

Det går också att definiera en egen funktion för validering:


<LiveCodeBlock scope={{TextField}}>
{`<TextField
  label={'Skriv in frukt'}
  validate={(value) => (value === 'frukt' ? true : 'Du måste skriva "frukt"!')}
/>`}
</LiveCodeBlock>


### Lösenord

Sätt `type={'password'}` för att kunna visa och dölja inmatad text.

<LiveCodeBlock scope={{TextField}}>
{`<TextField 
  type={'password'} 
  name={'pw'} 
  label={'Skriv in ditt lösenord'} 
  autoComplete={'off'}
/>
`}
</LiveCodeBlock>

### Desktop
Anpassa längden på inputfältet, tex med [Grid](./grid.mdx) eller [Flex](./flex.mdx), så att det matchar förväntad längd på det användaren ska mata in. Ett svenskt postnummer är alltid 5 siffror och då bör fältet vara kort medan ett fält där användaren ska fylla i alla sina namn behöver vara långt

    </GridItem>

</LiveCodeBlock>
