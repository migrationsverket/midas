Utfällbar komponent som används för att minska informationsmängden som direkt presenteras för användaren.

```tsx
<Accordion>
  {data.map(({ description, name, value }) => (
    <AccordionItem
      key={value}
      id={value}
      title={name}
    >
      {description}
    </AccordionItem>
  ))}
</Accordion>
```

<div className='card'>

      <AccordionItem
        key={value}
        id={value}
        title={name}
      >
        {description}
      </AccordionItem>
    ))}
  </Accordion>
</div>

## Egenskaper

Accordion har flertalet varianter och egenskaper.

### Contained

En tydligare variant av layouten när det finns en tydlig gräns mellan varje sektion.

```tsx
// highlight-start
<Accordion variant='contained'>
  // highlight-end
  {data.map(({ description, name, value }) => (
    <AccordionItem
      key={value}
      id={value}
      title={name}
    >
      {description}
    </AccordionItem>
  ))}
</Accordion>
```

<div className='card'>
  <Accordion variant='contained'>
    {fruits.shuffled().map(({ description, name, value }) => (
      <AccordionItem
        key={value}
        id={value}
        title={name}
      >
        {description}
      </AccordionItem>
    ))}
  </Accordion>
</div>

### Flera öppna samtidigt

En accordion har som standard bara en öppet samtidigt och stänger automatiskt andra sektioner när en ny öppnas. Detta kan ändras genom att sätta `type` till `multiple`.

```tsx
// highlight-start
<Accordion type='multiple'>
  // highlight-end
  {data.map(({ description, name, value }) => (
    <AccordionItem
      key={value}
      id={value}
      title={name}
    >
      {description}
    </AccordionItem>
  ))}
</Accordion>
```

<div className='card'>
  <Accordion type='multiple'>
    {fruits.shuffled().map(({ description, name, value }) => (
      <AccordionItem
        key={value}
        id={value}
        title={name}
      >
        {description}
      </AccordionItem>
    ))}
  </Accordion>
</div>

### Öppen som standard

En accordion kan ha en eller flera sektioner öppna som standard. Detta görs genom att sätta `defaultExpandedKeys` på `Accordion`. Detta är en array med nycklar för de sektioner som ska vara öppna som standard. Nycklarna bestäms med `id` på respektive `AccordionItem`.

```tsx
<Accordion
  type='multiple'
  // highlight-start
  defaultExpandedKeys={[2]}
  // highlight-end
>
  {data.map(({ description, name, value }, i) => (
    <AccordionItem
      key={value}
      id={i}
      title={name}
    >
      {description}
    </AccordionItem>
  ))}
</Accordion>
```

<div className='card'>

        id={i}
        title={name}

        {description}

</div>
