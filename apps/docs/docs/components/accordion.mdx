Utfällbar komponent som används för att minska informationsmängden som direkt presenteras för användaren.

<LiveCodeBlock hideCode scope={{ Accordion, AccordionItem, AccordionTrigger, AccordionContent }}>

    {fruits.shuffled().map(fruit => (
      <AccordionItem key={fruit.value} value={fruit.value}>
        <AccordionTrigger>{fruit.name}</AccordionTrigger>
        <AccordionContent>
          {fruit.description}
        </AccordionContent>
      </AccordionItem>
    ))}
</Accordion>`}
</LiveCodeBlock>

## Installation

```bash npm2yarn
npm install @midas-ds/components
```

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from '@midas-ds/components'
```
