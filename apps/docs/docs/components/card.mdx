Kortkomponenten är en yta med plats för information och åtgärder relaterat till ett specifikt ämne. Kort används primärt för att presentera innehåll på ett visuellt tilltalande sätt samt länka till innehållet.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/card
```

```tsx
import { Card } from '@midas-ds/card'
```
