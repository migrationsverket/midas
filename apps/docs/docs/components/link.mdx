Komponenten Link skapar en länk som kan vara i ett textstycke eller fristående.

```jsx
import { Link } from '@midas-ds/components'
```

```tsx
<Text>
  En text med en <Link href='#'>länk</Link> i ett textstycke.
</Text>
```

<div className='card'>
  <Text>
    En text med en <Link href='#'>länk</Link> i ett textstycke.
  </Text>
</div>

## Varianter

### Länk

```tsx {2}
<Text>
  En text med en <Link href='#länk'>länk</Link> i ett textstycke.
</Text>
```

<div className='card'>
  <Text>
    En text med en <Link href='#länk'>länk</Link> i ett textstycke.
  </Text>
</div>

### Fristående

Använd `standalone` för att använda komponenten som en fristående länk under t.ex ett textblock.

```tsx {3}
<Text>Text</Text>
<Link
  standalone
  href="#fristående"
>
  En länk som är fristående från text
</Link>
```

<div className='card'>
  För att visa en länk som en fristående komponent, använd propertyn <code>standalone</code>. Detta kan vara användbart
  när du vill att länken ska stå ut från omgivande text eller när du vill placera den på en egen rad.
  <Link
    standalone
    href='#fristående'
  >
    {'En länk som är fristående från text'}
  </Link>
</div>

### Stretched

Använd `stretched` för att låta hela föräldraelementet vara klickbart till länken. Den klickbara ytan placeras med `position: absolute` och sträcker sig över hela föräldraelementet. Placera därför `position: relative` på relevant element för att inte den klickbara ytan ska sträckas för långt.

```tsx {1,5,10}
<div style={{ position: 'relative' }}>
  <p>Text</p>
  <Link
    standalone
    stretched
    href='#stretched'
  >
    En länk som sträcker sig över hela ytan
  </Link>
</div>
```

<div className='card'>
  <div
    style={{
      position: 'relative',
      background: semantic.layer01Base,
      padding: '1rem',
    }}
  >
    En länk kan sträcka sig över en hel yta genom propertyn <code>stretched</code>. Detta kan vara användbart när du
    vill att hela ytan ska vara klickbar, till exempel i en kort eller en panel.
    <Link
      standalone
      stretched
      href='#stretched'
    >
      {'En länk som sträcker sig över hela ytan'}
    </Link>
  </div>
</div>

### Ny flik

Om en länk är satt till att öppnas i en ny flik förtydligas detta med en ikon. Detta görs genom att sätta `target='_blank'` på länken.

```tsx {3}
<Link
  href='#länk'
  target='_blank'
>
  En länk till en annan URL
</Link>
```

<div className='card'>
  <Link
    href='#länk'
    target='_blank'
  >
    {'En länk till en annan URL'}
  </Link>
</div>

### Nedladdning

Även vid nedladdning förtydligas detta med en ikon. Detta görs genom att sätta `download` på länken.

```tsx {3}
<Link
  href='#nedladdning'
  download
>
  Ladda ned senaste rapporten
</Link>
```

<div className='card'>
  <Link
    href='#nedladdning'
    download
  >
    {'Ladda ned senaste rapporten'}
  </Link>
</div>
