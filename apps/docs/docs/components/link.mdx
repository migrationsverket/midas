Komponenten Link skapar en länk som kan vara i ett textstycke eller fristående.

```tsx title=Link
<p>
  I vår
  <Link href='/changelog'>Changelog</Link>
  dokumenterar vi de senaste ändringarna i komponenterna.
</p>
```
<div className={'card'}>
  <p>
    I vår <Link href='/changelog'>Changelog</Link> dokumenterar vi de senaste ändringarna i komponenterna.
  </p>
</div>


## Installation

```bash npm2yarn
npm install @midas-ds/components
```

```jsx
import { Link } from '@midas-ds/components'
```

## Varianter

### Länk

```tsx title="default"
<p> Designsystemet utvecklas ständigt,
//highlight-start
  <Link href='/changelog'>vilket du kan kan se i vår Changelog</Link>
//highlight-end
Vi släpper kontinuerligt nya versioner med buggfixar, nya komponenter och nya funktioner.
</p>
```
<div className={'card'}>
  <p>Designsystemet utvecklas ständigt,<Link href='/changelog'>vilket du kan kan se i vår Changelog</Link>.
    Vi släpper kontinuerligt nya versioner med buggfixar, nya komponenter och nya funktioner.</p>
</div>

### Fristående

Använd `standalone` för att använda komponenten som en fristående länk under t.ex ett textblock.

```tsx title="standalone"
<p>Designsystemet utvecklas ständigt. Vi släpper kontinuerligt nya versioner med buggfixar,
nya komponenter och nya funktioner på befintliga komponenter.</p>
<Link
//highlight-start
  standalone
//highlight-end
  href="/changelog">
Se vad som har ändrats i senaste versionen
</Link>
```
<div className={'card'}>
  <Link standalone href="/changelog">Se vad som har ändrats i senaste versionen</Link>
</div>

### Stretched

Använd `stretched` för att låta hela föräldraelementet vara klickbart till länken. Föräldern måste ha `position: relative;` för att länkens klickyta inte ska gå för långt.

```tsx title="stretched"
<div
  style={{position: 'relative'}}
>
  <p>
    Designsystemet utvecklas ständigt. Vi släpper kontinuerligt
    nya versioner med buggfixar, nya komponenter och nya
    funktioner på befintliga komponenter.
  </p>
  <Link
    standalone
    //highlight-start
    stretched
    //highlight-end
    href='/changelog'
  >
    Se vad som har ändrats i senaste versionen
  </Link>
</div>
```
<div className={'card'}>
<div
  style={{
    position: 'relative',
    background: semantic.layer02,
    padding: '1rem',
  }}
>
  <p>
    Designsystemet utvecklas ständigt. Varannan vecka kommer det ett nytt släpp med buggfixar, nya komponenter eller nya
    funktioner på befintliga komponenter. Om du håller muspekaren här så kan du klicka på länken över hela den gråa
    ytan.
  </p>
  <Link standalone stretched href='#'>Läs mer om designsystemets nyheter</Link>
</div>
</div>
