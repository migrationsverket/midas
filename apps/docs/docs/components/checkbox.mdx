### Flera checkboxar

Använd `<CheckboxGroup>` för att gruppera flera checkboxar som hör samman.

```tsx
<CheckboxGroup
  label='Godkänner du våra villkor?'
  description='Läs texten först'
>
  <Checkbox value='1'>Jag har läst texten</Checkbox>
  <Checkbox value='2'>Jag godkänner villkoren</Checkbox>
</CheckboxGroup>
```

<div className='card'>
  <CheckboxGroup
    label='Godkänner du våra villkor?'
    description='Läs texten först'
  >
    <Checkbox value='1'>Jag har läst texten</Checkbox>
    <Checkbox value='2'>Jag godkänner villkoren</Checkbox>
  </CheckboxGroup>
</div>

## Välj alla

Ibland kan det vara lämligt att ge användaren möjlighet att snabbt kryssa i alla kryssrutor. Denna funktion finns inbyggt i `<CheckboxGroup>` via egenskapen `showSelectAll`. Om detta används på en sida med paginerat innehåll så skall endast de kryssrutor som finns på den aktuella sidan väljas.

```tsx
<CheckboxGroup
  label={'Godkänner du våra villkor'}
  description={'Läs texten först'}
  // highlight-start
  showSelectAll
  // highlight-end
>
  <Checkbox
    value='1'
    // highlight-start
    isRequired
    // highlight-end
  >
    Jag har läst texten
  </Checkbox>
  <Checkbox
    value='2'
    // highlight-start
    isRequired
    // highlight-end
  >
    Jag godkänner villkoren
  </Checkbox>
</CheckboxGroup>
```

<div className='card'>
  <CheckboxGroup
    label={'Godkänner du våra villkor'}
    description={'Läs texten först'}
    showSelectAll
  >
    <Checkbox
      value='1'
      isRequired
    >
      Jag har läst texten
    </Checkbox>
    <Checkbox
      value='2'
      isRequired
    >
      Jag godkänner villkoren
    </Checkbox>

  </CheckboxGroup>
</div>

### Kontrollerat val av Kryssruta

Om du vill kontrollera

```tsx
import React from 'react'
```

```tsx
const [isChecked, setIsChecked] = useState(true)

return (
  <Checkbox
    isSelected={isChecked}
    onChange={(checked: boolean) => setIsChecked(checked)}
  >
    Jag godkänner villkoren
  </Checkbox>
)
```

<div className='card'>
  <ControlledExample />
</div>

## Riktlinjer

- Om det är många alternativ så bör [Select](/components/select) användas istället.
- Använd inte kryssruta om användaren väntar sig att valet ska få effekt direkt
- Fältetikett ska inledas med en stor bokstav och inte följas av punkt.
