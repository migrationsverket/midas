## Varianter

### Flerval

Använd egenskapen `selectionMode="multiple"` för att slå på flervalsläget.

<LiveCodeBlock
  scope={{ Select }}
  {...props}
>
  {`
    <Select
      label='Vilka är dina favoritfrukter?'
      placeholder='Välj en eller flera frukter'
      description='Beskrivning'
      selectionMode='multiple'
      options={fruits.map(({ value, name }) => ({ id: value, name }))}
    />
  `}
</LiveCodeBlock>

### Välj alla

Egenskapen `isSelectableAll` kan användas för att lägga till en "Välj alla"-knapp.

<LiveCodeBlock
  scope={{ Select }}
  {...props}
>
  {`
    <Select
      label='Vilka är dina favoritfrukter?'
      placeholder='Välj en eller flera frukter'
      description='Beskrivning'
      selectionMode='multiple'
      options={fruits.map(({ value, name }) => ({ id: value, name }))}
      isSelectableAll
    />
  `}
</LiveCodeBlock>

## Sektioner

Strukturera om alternativen i nestade sektioner genom att strukturera options på följande sätt

```ts
const options = [
  {
    name: 'Frukter',
    children: [
      { id: 'kiwi', name: 'Kiwi' },
      { id: 'banana', name: 'Banan' },
      { id: 'apple', name: 'Äpple' },
    ],
  },
  {
    name: 'Grönsaker',
    children: [
      { id: 'carrot', name: 'Morot' },
      { id: 'broccoli', name: 'Broccoli' },
    ],
  },
]
```
