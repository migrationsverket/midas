## Introduktion

Select är en typ av inmatningsfält som används för att välja **ett** alternativ från en uppsättning av flera alternativ. Om det ska gå att välja flera eller inget alternativ är det [MultiSelect](./multi-select.mdx) eller [Checkbox](./checkbox.mdx) som ska användas.
Om det är färre alternativ än fem används med fördel [Radio](./radio.mdx) istället.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/select
```

```tsx
import { Item, Select } from '@midas-ds/select'
```
