## Introduktion

En Select visar en hopfällbar lista med alternativ och låter en användare välja ett av dem.

Select är en typ av input-fält som används för att välja **ett** alternativ av fler än fyra (4) av flera alternativ. Om det ska gå att välja flera eller inget alternativ är det [Checkbox](./checkbox.mdx) som ska användas
Om det är fler alternativ än färre än fyra (4) används med fördel [Radio](./radio.mdx) istället.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/select
```

```tsx
import { Item, Select } from '@midas-ds/select'
```
