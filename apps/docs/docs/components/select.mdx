## Varianter

### Flerval

Använd egenskapen `selectionMode="multiple"` för att slå på flervalsläget.

<LiveCodeBlock
  scope={{ Select }}
  {...props}
>
  {`
    <Select
      label='Vilka är dina favoritfrukter?'
      placeholder='Välj en eller flera frukter'
      description='Beskrivning'
      selectionMode='multiple'
      options={fruits.map(({ value, name }) => ({ id: value, name }))}
    />
  `}
</LiveCodeBlock>

#### Välj alla

Egenskapen `isSelectableAll` kan användas för att lägga till en "Välj alla"-knapp.

<LiveCodeBlock
  scope={{ Select }}
  {...props}
>
  {`
    <Select
      label='Vilka är dina favoritfrukter?'
      placeholder='Välj en eller flera frukter'
      description='Beskrivning'
      selectionMode='multiple'
      options={fruits.map(({ value, name }) => ({ id: value, name }))}
      isSelectableAll
    />
  `}
</LiveCodeBlock>

#### Tags

Egenskapen `showTags` kan användas för att visa valda alternativ som taggar.

<LiveCodeBlock
  scope={{ Select }}
  {...props}
>
  {`
    <Select
      label='Vilka är dina favoritfrukter?'
      placeholder='Välj en eller flera frukter'
      description='Beskrivning'
      selectionMode='multiple'
      options={fruits.map(({ value, name }) => ({ id: value, name }))}
      showTags
    />
  `}
</LiveCodeBlock>
