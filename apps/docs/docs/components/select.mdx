Select är en typ av inmatningsfält som används för att välja **ett** alternativ från en uppsättning av flera förbestämda alternativ.

<LiveCodeBlock hideCode scope={{ Select, Item }}>
  {`<Select
    label="Vad är din favoritfrukt?"
    placeholder="Välj en frukt"
    items={fruits.map(fruit => {return {id: fruit.value, name: fruit.name}})}
>
    {(item) => <Item>{item.name}</Item>}
</Select>`}
</LiveCodeBlock>

## Installation

```bash npm2yarn
npm install @midas-ds/components
```

```tsx
import { Item, Select } from '@midas-ds/components'
```


## Riktlinjer

 Om det är färre alternativ än fem bör [Radio](./radio.mdx) användas istället.
