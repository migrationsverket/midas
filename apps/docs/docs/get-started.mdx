## Designer

För dig som designar gränssnitt finns ett UI-kit till Adobe XD. Det innehåller färdiga designkomponenter och layoutmallar som följer Migrationsverkets digitala stil och grafiska profil.

- Här kan du ladda ner vårt <a target="\_blank" href={ require("/files/MIDAS_UIkit_17.zip").default } download>UI-kit till Adobe XD (version 17)</a>
- Om du inte har Adobe XD kan se hur komponenterna ser ut via [onlineversionen](https://xd.adobe.com/view/95ad014c-2540-44e2-9adf-5ad4c8751bbd-c0fe/?hints=off)

## Utvecklare

Installera komponenter via `npm install @midas-ds/components`.

### Importera och använd komponent

Generellt, importera och använd direkt:

```tsx
import { Component } from '@midas-ds/components';
...
<Component/>
```

För specfika instruktioner, se respektive komponent.

### Installera Inter som typsnitt

Detta är ett rekommenderat sätt att installera vårt typsnitt Inter. Det går självklart att använda andra metoder så länge vikterna 400, 500 och 600 inkluderas.

```tsx {1} title="App.tsx (rootfilen i din app)"
import '@midas-ds/components/global.css'

export default function App({ children }) {
  return <main>{children}</main>
}

export default App
```

Mer info finns på [GitHub](https://github.com/migrationsverket/midas) samt under [guider](/dev/common-issues)
