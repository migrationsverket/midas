
Importera `default.css`. Denna fil innehûËller global css fûÑr att layout-komponenterna ska fûË rûÊtt utseende. Importera filen i roten av din applikation:

## Rekommenderade mûÑnster

### Bara Header

Detta mûÑnster passar fûÑr applikationer som inte har nûËgon navigation eller dûÊr det finns max 3 lûÊnkar.

### Header och Sidebar

[Header](./components/header) med [Sidebar](./components/sidebar) ûÊr ett av det vanligaste kombinationerna i vûËra applikationer. Det ger mûÑjlighet att ha mûËnga navigationslûÊnkar i Sidebar och handlingar (tex logga ut och instûÊllningar) i Header

### Header, Sidebar och Panel

Kombinationen [Header](./components/header), [Sidebar](./components/sidebar) och [Panel](./components/panel) det mest kompletta mûÑnstret. Det ûÊr det vanligaste fûÑr vûËra mer avancerade applikationer, dûÊr Panel anvûÊnds fûÑr att visa kontextuell information t.ex. notifieringar eller mer infomation om element i en lista.

### Header och Navbar

Detta mûÑnster ûÊr reserverat fûÑr applikationer som ska upplevas som mobilappar t.ex. Min Sida och inte tûÊnkt fûÑr interna desktopapplikationer. HûÊr ersûÊtts Sidebar med [Navbar](./components/navbar) som ligger lûÊngst ner pûË sidan. [Navbar](./components/navbar) har endast plats fûÑr ett begrûÊnsat antal (max 5) menyalternativ eftersom det inte fûËr plats fler pûË en liten skûÊrm.

### Header, Navbar och Panel

Analogt med mûÑstret fûÑr Header, Sidebar och Panel men Sidebar ûÊr utbytt mot [Navbar](./components/navbar).

```tsx
  Layout,
  LayoutContent,
  Main,

              <NavigationLink
                href='/installningar'
                icon={<Settings />}
              >
                InstûÊllningar
              </NavigationLink>
            </NavigationItem>
          </Navigation>
        </Sidebar>

        <Main>{/* Ditt innehûËll */}</Main>
      </LayoutContent>
    </Layout>
  )
}
```
