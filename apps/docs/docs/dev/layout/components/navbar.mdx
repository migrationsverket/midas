# Navbar

Navigationskomponent som används som ersättare för [Sidebar](/dev/layout/components/sidebar). `Navbar` renderar ett semantiskt `<footer>`-element.

## Användning

Syftet med Navbar är att få applikationen att likna en native mobilapp, som oftast har navigation i botten av appen. Navbar ska inte användas tillsammans med Sidebar, utan Navbar ersätter Sidebar som applikationens huvudnavigation.

### Använd Navbar:

- När applikationen ska upplevas som en mobilapp.

### Använd inte Navbar:

- Om applikationen behöver mer än 5 menyalternativ i navigationen, eftersom det gör det för trångt på små skärmar.
- Om applikationen behöver undermenyer eller komplex navigering.

## Exempel

```tsx
import { Navbar, Navigation, NavigationItem, NavigationLink } from '@midas-ds/layout'
import { House, Search, User } from 'lucide-react'

export default function App() {
  return (
    <Navbar>
      <Navigation>
        <NavigationItem>
          <NavigationLink
            href='/'
            icon={<House />}
            isActive
          >
            Hem
          </NavigationLink>
        </NavigationItem>
        <NavigationItem>
          <NavigationLink
            href='/sok'
            icon={<Search />}
          >
            Sök
          </NavigationLink>
        </NavigationItem>
        <NavigationItem>
          <NavigationLink
            href='/profil'
            icon={<User />}
          >
            Profil
          </NavigationLink>
        </NavigationItem>
      </Navigation>
    </Navbar>
  )
}
```
