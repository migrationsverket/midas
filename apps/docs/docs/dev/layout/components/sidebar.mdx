# Sidebar

Sidebar är huvudnavigationen i layoutpaketet. På mobil döljs Sidebar automatiskt. Använd `MobileMenu` i `Header` för mobilnavigering.

## Grundläggande användning

```tsx
import { Sidebar, Navigation, NavigationItem, NavigationLink } from '@midas-ds/layout'
import { House, Settings } from 'lucide-react'

export default function App() {
  const pathname = usePathname() // or equivalent for your router

  return (
    <Sidebar title='Min applikation'>
      <Navigation>
        <NavigationItem>
          <NavigationLink
            href='/'
            icon={<House />}
            isActive={pathname === '/'}
          >
            Hem
          </NavigationLink>
        </NavigationItem>
        <NavigationItem>
          <NavigationLink
            href='/installningar'
            icon={<Settings />}
            isActive={pathname === '/installningar'}
          >
            Inställningar
          </NavigationLink>
        </NavigationItem>
      </Navigation>
    </Sidebar>
  )
}
```

## Ihopfällt läge

Sidomenyn kan fällas ihop ihop till enbart ikoner. Det kan styras antingen okontrollerat (internt tillstånd) eller kontrollerat av föräldrakomponenten.

### Okontrollerat

```tsx
<Sidebar
  title='Min applikation'
  defaultCollapsed={false}
>
  ...
</Sidebar>
```

### Kontrollerat

```tsx
const [isCollapsed, setIsCollapsed] = useState(false)

<Sidebar
  title='Min applikation'
  isCollapsed={isCollapsed}
  onCollapseChange={setIsCollapsed}
>
  ...
</Sidebar>
```

:::note

När sidomenyn är kollapsad visas etiketten i `NavigationLink` som en tooltip vid hover. Se till att varje länk alltid har en `children`-prop (etikett) eftersom det är nödvändigt för tillgänglighet.

:::

## Props

| Prop               | Typ                              | Standard | Beskrivning                                              |
| ------------------ | -------------------------------- | -------- | -------------------------------------------------------- |
| `title`            | `string`                         | —        | Titel som visas högst upp i sidomenyn (dold vid kollaps) |
| `isCollapsed`      | `boolean`                        | —        | Kontrollerat kollapsläge                                 |
| `defaultCollapsed` | `boolean`                        | `false`  | Starttillstånd för okontrollerat läge                    |
| `onCollapseChange` | `(isCollapsed: boolean) => void` | —        | Callback vid kollaps/expansion                           |
| `children`         | `ReactNode`                      | —        | Sidomenysinnehåll (typiskt `Navigation`)                 |
