## Migrera från 6.x.x till 7.0.0

`TextField` stödjer inte längre `validationType`.

### pattern

Om du vill använda HTML native constraint validation, använd egenskapen `pattern`.
Läs mer om [pattern på MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/input#pattern)

### validate

För anpassad validering med tillhörande felmeddelande, använd `validate`.
Läs mer om egenskapen under vår [dokumentation för TextField](/components/textfield/#egen-validering)
