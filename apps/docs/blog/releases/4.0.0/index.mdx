## Migrera från 3.x.x till 4.0.0

### Select

`MultiSelect` ersätts av `Select`, vänligen använd `selectionMode` för en flerväljare:

```tsx
<Select selectionMode='multiple' />
```

Se mer information om [hur du använder Select](/components/select)
