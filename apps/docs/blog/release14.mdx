## Ny komponent: Modal

Modalen öppnar en ruta ovanför gränssnittet som användaren måste interagera med innan den kan gå vidare. Modalen har flera användningsområden, t.ex. för att förtydliga konsekvenserna av ett val i gränssnittet eller presentera mer information. Läs mer på komponentsidan för [Modal](/components/modal).

```bash npm2yarn
npm install @midas-ds/modal
```

## Ikoner på Button

Button kan nu acceptera ikoner som prop. Ikonen läggs till via `icon`.
Du kan välja vilken sida av texten ikonen ska vara på med `iconPlacement`.

<LiveCodeBlock
  scope={{
    Plus,
    Button
  }}
>
  {`<Button
    icon={Plus}
    variant='tertiary'
    iconPlacement="right"
  >
    {"Lägg till rad"}
  </Button>
`}
</LiveCodeBlock>
