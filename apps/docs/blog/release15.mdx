## Uppdatering av Link

Komponenten <Link href='/components/link'>Link</Link> är nu uppdaterad med varianterna Fristående och Utsträckt. 

### Fristående länk 
Fristående länk används när länken ligger utanför ett textstycke. 

<LiveCodeBlock scope={{ Link }}>
  {`<>
    <p>Designsystemet utvecklas ständigt. Varannan vecka kommer det ett nytt släpp med buggfixar, nya komponenter eller nya funktioner på befintliga komponenter.</p>
    <Link 
        standalone 
        href="/blog"
    >
        Läs mer om designsystemets nyheter
    </Link>
</>`}
</LiveCodeBlock>


### Utsträckt länk 

Utsträckt länk används för att göra hela föräldraelementet klickbart.

Vi har därför skapat ett designmönster som stöd för att välja rätt komponent.
