## Migrera från 2.x.x till 3.0.0

### SearchField

`SearchField` stödjer inte längre `isRequired`. Beteendet kan efterliknas med hjälp av `validate`

```tsx
<SearchField validate={v => !!v.length || 'Var god ange åtminstone ett tecken' />
```

Se mer information om [hur du använder SearchField](/components/form/search-field)

Se alla förändringar i detalj på [changelog](https://github.com/migrationsverket/midas/releases/tag/v3.0.0).
