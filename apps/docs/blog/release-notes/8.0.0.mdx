## Migrera från 7.x.x till 8.0.0

### Select

Värdet i callbacken `onSelectionChange` för komponenten `Select` har ändrats.
Numera skickas valda nycklar med en `Selection` type (`"all" | Set<string | number>`) när användaren justerar sitt val.

```diff
-   onSelectionChange?: ((value: Selection | Key | Key[]) => void) | undefined
+   onSelectionChange?: ((keys: Selection) => void) | undefined
```

Detta innebär att du behöver hantera om användaren klickar på "Välj alla" manuellt:

```tsx
const handleSelectionChange = (keys: Selection) => {
  if (keys === 'all') {
    return setSelection(new Set(options.map(({ id }) => id)))
  }
  return setSelection(keys)
}
```

För mer information, läs [instruktioner för `Select` och kontrollerade val](/components/select/#kontrollerade-val)

Se alla förändringar i detalj på [changelog](https://github.com/migrationsverket/midas/releases/tag/v8.0.0).
